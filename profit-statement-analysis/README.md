# Profit Statement Analysis — 利润表财务分析

> 九星 Apple APR 授权经销商利润表分析 — 毛利率/费用率/净利率计算、门店评分诊断、行业对标、HTML 报告输出。

## 核心特征

| 特征 | 说明 |
|------|------|
| 数据源 | 固定结构利润表 Excel（template 工作表） |
| 门店数 | 6 家 Apple APR 门店 |
| 分析维度 | 收入分析 / 盈利能力 / 费用结构 / 门店诊断 / 行业对标 |
| 环比对比 | 自动计算月度环比增长率 |
| 输出格式 | Electric Studio 风格单文件 HTML 报告 |
| 输出路径 | 与利润表同目录 |

## 适用场景

- 月度/季度利润表更新后自动分析
- 门店经营诊断与评分
- 行业基准对标
- 经营复盘报告生成
