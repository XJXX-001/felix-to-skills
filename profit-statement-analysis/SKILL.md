## 分析框架

### 1. 收入分析
- 月度总收入趋势（柱状图）
- 各店收入贡献排名（水平柱状图）
- 各月各店收入明细表

### 2. 盈利能力
- 综合毛利率 = 毛利/收入 × 100%
- 各店毛利率排名，标注高于/低于行业
- 净利率 = 净利润/收入 × 100%
- 盈利门店 vs 亏损门店划分
- 月度利润走势

### 3. 费用结构
- 工资占比 = 工资/收入 × 100%
- 房租占比 = 房租/收入 × 100%
- 费用效率 = 毛利/销售费用（>1 健康，<1 警示）
- O2O 平台手续费分类汇总
- 异常支出标记（如办公用品突增、广告费异常）

### 4. 门店诊断（A-D 级评分）
- **A级（润达APR）**: 净利率 >1%，费用率 <4.5%，核心利润引擎
- **B+级（天虹APR）**: 净利率 0.5-1%，费用率 4.5-5.5%
- **B-级（九江铜锣湾）**: 净利率 0-0.5%，费用率 5-6%
- **C级（梦想/龙南）**: 亏损或微利，费用率 6-7%
- **D级（南百）**: 连续亏损，费用率 >8%，亟需整改

### 5. 行业对标（Apple APR 授权经销商基准）

| 指标 | 行业均值 | 行业优异 | 说明 |
|------|----------|----------|------|
| 毛利率 | 5-8% | 8-10% | 苹果硬件毛利低，依赖规模 |
| 净利率 | 1-3% | 3-5% | 优质门店可达 3%+ |
| 销售费用率 | 4-7% | 3-4% | 含人工+房租+运营 |
| 房租占比 | 3-5% | 1.5-3% | 商场店租金较高 |
| 人工占比 | 4-6% | 2-4% | 取决于人效 |
| O2O费占比 | 0.3-0.8% | <0.3% | 京东/美团等平台抽成 |
| AppleCare+渗透率 | 15-30% | 30%+ | 未单独核算的需推动 |
| 配件收入占比 | 8-15% | 15%+ | 高毛利品类 |
| 二手机回收 | 5-10% | 10%+ | 零成本启动的利润补充 |

### 6. 环比分析
- 月度收入环比增长率
- 月度利润环比增长率
- 各店与上期对比波动
- 季节性预警（如3月全线下滑）

## 输出格式

使用 **Electric Studio** 风格 HTML 报告（R2 数据分析类型），7个章节：

1. 执行摘要 — 4个核心KPI卡片 + 积极信号/风险关注
2. 收入分析 — 月度趋势柱状图 + 各店贡献 + 明细表
3. 盈利能力 — 毛利率/净利率排名 + 月度利润走势
4. 费用结构 — 费用构成 + O2O手续费 + 异常支出标记
5. 门店诊断 — 6店评分卡 + 关键经营指标汇总表
6. 行业对标 — 9项指标对比表 + 优势/改进空间
7. 总结建议 — 短期/中期/长期行动项

## 注意事项

1. 管理费用（行38-49）可能全为0，所有费用已归入销售费用
2. 所得税费用可能为0，净利润=利润总额
3. 返利收入可能为0，需确认是否按季度/年度结算
4. 赣州龙南是唯一有服务收入的门店
5. 南百苹果店的「办公用品」和「广告宣传费」需定期监控异常
6. 所有金额保留2位小数，以万元显示时注意千位分隔符
